🥟 DumplingThoughts

Personal Second Brain — Telegram Bot + Dashboard
Mar 2026 Live

My thoughts used to live everywhere — Telegram messages to myself, multiple open browser tabs, a notes app graveyard. So I built a second brain to track everything.

DumplingThoughts is a Telegram bot that captures anything I send it — reflections, to-dos, Thai vocabulary, links, photos — categorises it with one tap, sends it to a Google Sheet, and visualizes it in a searchable web dashboard. It runs 24/7 on Railway so it works whether my laptop is open or not.

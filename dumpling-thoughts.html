🧩 The Problem

As someone on a sabbatical juggling career development, language learning, personal reflection, and constant curiosity — my thoughts lived everywhere. Ideas in Telegram messages to myself. Buried in multiple browser tabs. Thai vocabulary disappearing into the notes app. To-dos spread across three different tools.

I had no single place where everything lived — and no way to look back at how my thinking had evolved over time. I needed a second brain. So I built one.

On top of that, I wasn't going to be sitting in front of a laptop all day, so I needed a solution that worked on the go and was accessible from my phone.

⚙️ How It Works

📲
Send anything to the Telegram bot A thought, link, photo, Thai word, to-do — whatever's on your mind
🗂
Choose a category with one tap 6 categories appear as inline buttons — no typing needed
⚡
Set priority + due date (To-Dos only) High / Medium / Low, then type a date or skip
🏷
Add an optional keyword tag e.g. "python", "thai", "career" — for filtering later
✅
Saved to Google Sheets forever With timestamp, category, priority, content, and keyword
📊
View anytime in the Streamlit dashboard Filter, search, visualise, and mark things done

🧠 What I Learned

🔐

Secrets Management — The Hard Way

Accidentally pushed API keys to a public GitHub repo early on. Immediately revoked everything, scrubbed the git history, and rebuilt security practices from scratch.

⚙️

Async Python

Telegram bots run on asyncio under the hood. Debugging async errors — especially Python 3.14 compatibility issues — taught me how the event loop works and why version pinning matters.

🌐

Multi-Environment Deployment

Getting identical code to run on Windows locally, Railway (Linux), and Streamlit Cloud (Linux) required understanding how each platform handles Python versions and secrets differently.

🏗

Building in Phases

Starting with raw capture only meant I had something working from day one. Each phase built on a stable foundation rather than trying to build everything at once.

🔌

API Limitations Are Real

Google service accounts can't upload to Drive due to storage quota restrictions. Pivoted to Cloudinary — a good lesson in reading API documentation before building.
